About

About the team and the creation of the application.

What?

Our project is to develop an application that will be available through a web browser to convert assembly code (68HC12 reduced instruction set) into C code, which is much easier to understand.

Why?

We choose this project because we want other students of computer engineering to have an easier time than we did learning assembly.

Features

During the output of the decompiled code, we want our application to display a code flow graph for a given assembly function. We want our application to store our usersâ€™ previously decompiled code.
